# LyricsX 2.0
An iTunes/VOX plugin to show lyrics on desktop.
1.0 is dropped for itsmaintainability, and 2.0 is in the developing stage.

### System Requirement
* Mac OS X 10.11 (El Capitan) or above

### Build Requirement
* Mac OS X 10.12 (El Capitan) or above
* Xcode 9.0 or above
* Swift 4.0 or above
